# Plan 69-01: Adaptive Sizing And High-Leverage Governor

## Objective

Implement a deterministic governor that scales or blocks candidate notional using
setup quality, market quality, stop/liquidation geometry, available balance,
manual margin pressure, and recent guard evidence.

## Must Haves

- Add an adaptive sizing result with transparent diagnostics.
- Apply the governor before AI so AI can only echo or veto the final setup.
- Keep final notional inside configured hard caps and stop-risk limits.
- Block or downsize unsafe high-leverage conditions using liquidation distance,
  stop distance, volatility/range, and liquidity/slippage proxies.
- Include manual exposure as margin pressure while keeping manual symbols out of
  bot-managed counts and exit actions.
- Add tests for sizing scale-up/down, high-leverage blocking, manual margin
  pressure, and agent diagnostics.
