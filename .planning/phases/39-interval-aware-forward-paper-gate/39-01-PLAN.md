# Plan 39-01: Interval-Aware Forward Paper Gate

## Goal

Promote only explicitly selected passing intervals toward forward-paper
observation while preserving strict all-interval promotion for any live resume
or risk-profile change.

## Scope

- Add a promotion scope to `ops strategy-promotion-check`.
- Preserve default all-interval behavior.
- Add selected-interval checks for one or more explicit intervals.
- Report selected interval summaries and cell checks.
- Ensure selected-interval success cannot be mistaken for live-resume
  permission.
- Verify the Phase 38 matrix with `quant_setup_selective` on `5m`.

## Out Of Scope

- No live timer restore.
- No risk profile apply.
- No exchange order, close, or position adjustment.
- No claim that a single passing interval proves profitability.

## Verification

- Focused promotion and CLI tests.
- Real Phase 38 matrix check for selected `5m`.
- Real Phase 38 matrix check for default all-interval scope.
- Full local test suite.
