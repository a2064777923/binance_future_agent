<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Export directory -> parser | Local runtime files can be malformed, huge, or noisy. |
| RSS/Atom XML -> feed parser | External XML content can contain unexpected structure. |
| Feed URL config -> fetcher | Future live URLs can point outside expected news sources. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-01 | Tampering | manual parser | mitigate | Parse JSON/JSONL structurally, surface malformed files, and keep deterministic file ordering. |
| T-03-02-02 | Information Disclosure | source config | mitigate | Do not read cookie/token/key files; tests must use only static fixtures and temporary export directories. |
| T-03-02-03 | Denial of Service | XML/manual parsing | mitigate | Keep fixture-driven parsers small, avoid recursive external entity handling, and leave scheduling/rate control out of Phase 3. |
</threat_model>
