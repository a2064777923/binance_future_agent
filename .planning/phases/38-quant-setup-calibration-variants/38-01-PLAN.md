# Plan 38-01: Quant Setup Calibration Variants

## Goal

Create a repeatable offline calibration path for deterministic setup rules so
the project can compare strategy profiles without changing live defaults.

## Scope

- Add an optional setup profile to deterministic setup generation.
- Keep the standard profile as the default for live code paths.
- Add built-in backtest variants for selective and scalp calibration.
- Expose profile settings in backtest output.
- Run a recent hot-symbol matrix comparing baseline, selective, and scalp
  quant setup variants.
- Run strategy promotion checks for each variant.

## Out Of Scope

- No live service/timer changes.
- No risk profile apply.
- No exchange order or position mutation.
- No claim of profitability unless promotion evidence passes.

## Verification

- Focused setup/backtest/matrix/CLI tests.
- Recent matrix smoke on public Binance market data.
- Promotion checks on the generated matrix.
- Full local test suite.
