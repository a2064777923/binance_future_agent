# Plan 60-01: Server Evidence And Pilot Resume Packet

<objective>
Deploy Phase 59 controls to the isolated server, verify local and server tests,
capture timer/service state before and after deploy, and produce a current
operator packet plus live-resume preview showing whether the pilot remains
fail-closed or is ready for separate operator confirmation.
</objective>

<tasks>

1. Preflight server state.
   - Read live/paper timer and service state.
   - Read current cap env values and manual symbol list.
   - Abort deploy if live or paper service remains active after timer pause.

2. Deploy scoped code update.
   - Pause only the BFA live/paper timers.
   - Sync current repository source/tests/docs into
     `/opt/binance-futures-agent/app`.
   - Reinstall editable package with `/opt/binance-futures-agent/.venv`.
   - Do not write outside `/opt/binance-futures-agent` or
     `/etc/binance-futures-agent`.

3. Verify server behavior.
   - Run focused tests for Phase 59 live-resume controls and CLI.
   - Run full server suite.
   - Run a read-only server health or smoke check if useful.
   - Restore live and paper timers.

4. Generate server evidence artifacts.
   - Generate current operator resume decision packet at
     `runtime/phase60-operator-decision.json`.
   - Generate current live-resume preview at
     `runtime/phase60-live-resume-plan.json`.
   - Record final timer/service state and current cap values.
   - Inspect recent live-runner evidence for active-position review,
     adjustment plan, entry-capacity preflight, and order trace identifiers
     where available.

5. Apply operator-requested cap widening.
   - Preserve `BFA_MANUAL_POSITION_SYMBOLS=BTWUSDT`.
   - Raise the active `30u_10x_multi_dynamic` target from 5 positions/50 USDT
     per-position notional/300 USDT portfolio notional to 6 positions/60
     USDT/360 USDT.
   - Keep per-trade risk and daily-loss caps unchanged.

6. Close GSD phase.
   - Write summary and verification.
   - Update ROADMAP, REQUIREMENTS, STATE, and PROJECT.
   - Commit the Phase 60 closeout.

</tasks>
