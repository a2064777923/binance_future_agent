# Plan 31-01: Active Position Review

## Objective

Add a read-only position review command that turns live exchange positions plus
local submitted-intent plans into deterministic recommendations.

## Tasks

- Extend submitted-intent hold evidence with entry, stop, and target prices.
- Add `bfa.ops.position_review` with:
  - `PositionReviewReport`
  - per-position recommendations
  - PnL percent, stop-risk R multiple, target progress, hold-time progress
- Add CLI command `python -m bfa.cli ops position-review`.
- Add tests for:
  - near-target long -> `trail_or_reduce`
  - expired hold window -> `close_review`
  - unprotected position -> urgent `close_review`
  - CLI JSON output
- Update README/GSD documentation.

## Safety

This phase is read-only. It does not place orders, cancel algo orders, modify
stops, or change the live risk profile.
