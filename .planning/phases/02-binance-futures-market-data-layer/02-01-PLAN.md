## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Binance REST -> client parser | External JSON and headers enter local code through public REST responses. |
| Configured base URL -> urllib transport | Environment-derived URL controls the remote public endpoint. |
| Fixture files -> tests | Static JSON fixtures are treated as untrusted external payload shapes. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-01-02 | Information Disclosure | `BinanceFuturesRestClient` | mitigate | Implement public unsigned calls only and assert tests do not send API-key, signature, listen-key, account, or order headers. |

| T-02-SC | Tampering | Python package dependencies | mitigate | No package-manager installs are authorized; keep `pyproject.toml` dependencies unchanged unless a later explicitly audited plan changes this. |
</threat_model>
