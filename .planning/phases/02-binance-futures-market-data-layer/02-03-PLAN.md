<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| WebSocket message -> parser | External live-stream JSON enters local event parsing. |
| Configured stream list -> URL builder | User/config-provided symbols and intervals control the public stream subscription URL. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-03-01 | Tampering | `parse_market_stream_message` | mitigate | Parse known event shapes into typed snapshots, preserve unknown payloads with context, and cover combined/raw fixtures. |
| T-02-03-02 | Information Disclosure | stream builder | mitigate | Reject listen-key/private/account/order streams and never include credentials in URLs. |
| T-02-03-03 | Denial of Service | reconnect/backoff helper | mitigate | Provide capped backoff hooks and keep actual long-running socket loops out of Phase 2. |
| T-02-SC | Tampering | Python package dependencies | mitigate | No WebSocket package install is authorized; implement pure parsing/building utilities with the standard library. |
</threat_model>
